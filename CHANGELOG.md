## 开发提示

- 在修改代码时，请考虑保持现有的设计风格和功能。
- 如果添加新功能，请确保它们与现有功能和设计协调一致。
- 在对 CSS 进行更改时，请注意保持响应式设计的完整性。
- 如果修改了 `_config.yml`，请确保更新相关的模板文件。
- 在添加新的 JavaScript 功能时，请考虑性能影响和浏览器兼容性。
- 每次添加新功能后，请更新开发日志，注明详细的开发内容
